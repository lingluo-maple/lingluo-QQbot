
# lingluo-QQbot

这是一个用Python编写的QQ机器人

### 目前功能
 - 入群欢迎
 - 手动禁言~~没什么用的功能~~
 - 获取机器人加入的群聊

#### 你需要在**config.yml**文件中修改配置

### Config

参数  | 说明
-------  | -----
Host | 你的QQ号
Robot | 机器人的QQ号
DebugGroup | 功能测试用群号
Group | 机器人运行的群号
Image | 需要发送的图片的路径
