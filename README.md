# lingluo-QQbot

一个基于go-cqhttp和Avilla-core的QQ机器人

~~(求个star)~~

## 目前功能

- Pixiv pid搜图
- Lolicon API色图
- 群员数据库 (记录qq号与发言时间)
- 以图搜图
- 获取当前群所有成员
- (半)手动发送消息
